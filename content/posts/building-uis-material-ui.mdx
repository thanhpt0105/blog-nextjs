## Creating a Custom Theme

One of MUI's most powerful features is its theming system. Here's how to create a custom theme:

```typescript
import { createTheme, ThemeProvider } from '@mui/material/styles';

const theme = createTheme({
  palette: {
    primary: {
      main: '#1976d2',
    },
    secondary: {
      main: '#dc004e',
    },
  },
  typography: {
    fontFamily: '"Roboto", "Helvetica", "Arial", sans-serif',
    h1: {
      fontSize: '3rem',
      fontWeight: 700,
    },
  },
});

function App() {
  return (
    <ThemeProvider theme={theme}>
      {/* Your app content */}
    </ThemeProvider>
  );
}
```

## Essential Components

### Layout Components

MUI provides several components for building layouts:

- **Container**: Centers content and provides responsive padding
- **Grid**: Flexible grid system for layouts
- **Box**: Utility component for spacing and styling
- **Stack**: One-dimensional layouts

### Common Components


## Dark Mode Support

Adding dark mode is straightforward with MUI:

```typescript
const theme = createTheme({
  palette: {
    mode: 'dark', // or 'light'
  },
});
```

You can toggle between light and dark modes dynamically by managing the mode state.

## Customization Techniques

### Overriding Component Styles

You can customize components using the `sx` prop:

```typescript
<Button
  sx={{
    borderRadius: 2,
    textTransform: 'none',
    '&:hover': {
      backgroundColor: 'primary.dark',
    },
  }}
>
  Custom Button
</Button>
```

### Global Theme Overrides

Customize all instances of a component:

```typescript
const theme = createTheme({
  components: {
    MuiButton: {
      styleOverrides: {
        root: {
          borderRadius: 8,
          textTransform: 'none',
        },
      },
    },
  },
});
```

## Best Practices

1. **Use Theme Values**: Reference theme values instead of hardcoding
2. **Responsive Design**: Use breakpoints for responsive layouts
3. **Consistent Spacing**: Use the theme's spacing system
4. **Accessibility**: Always include proper labels and ARIA attributes
5. **Tree Shaking**: Import components individually for smaller bundles

## Conclusion

Material-UI is a powerful tool for building modern React applications. Its comprehensive component library, theming capabilities, and focus on accessibility make it an excellent choice for projects of any size.
